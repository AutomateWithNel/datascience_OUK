# Naming Rules

## Folders
- Use official course code first: `COURSE 123 Course Title`
- Use `Year N/Semester N` for academic grouping

## Source Files
- Format: `CourseCode - Module NN - Topic.ext`
- Example: `COURSE 123 - Module 03 - Topic.pdf`

## Notes
- Course note: `Module 03 - Normalization.md`
- Concept note: `Third Normal Form.md`
- Revision note: `Midterm Revision.md`

## Assignments and Reports
- Quiz file: `Module 03 - Quizzes.md`
- Practical task: `Module 03 - Practical Task.md`
- Practical script: `Module03Practical.py`
- Case study folder: `Case Studies/Module 03 - Case Study Topic`
- Case study details/question file: `Case Study Details - Module 03 - Topic.ext`
- Case study report/submission: keep the original meaningful title when already submitted; otherwise use `Final Submission - Module 03 - Topic.md` or `Report - Module 03 - Topic.docx`.

## Avoid
- Generic names like `Paper.pdf`, `module7.pdf`, `notes.docx`
- Duplicate naming styles inside the same course
