# Nelson OUK Study Vault

## Purpose
This vault stores academic source materials and study notes with the rule that course files are the first reference point before outside knowledge.

## Structure
- `01 Academic` holds official course material by year, semester, and course.
- `02 Concepts` holds topic notes that connect ideas across courses.
- `03 Cross-Course Resources` holds timetables, general documents, and materials not owned by one course.
- `99 Inbox` holds files that still need classification.

## Working Rule
For each course:
1. Keep original PDFs, slides, and documents in `01 Source Materials`.
2. Write your understanding in `02 Notes`.
3. Track submissions in `03 Assignments`.
4. Build exam prep in `04 Revision`.
