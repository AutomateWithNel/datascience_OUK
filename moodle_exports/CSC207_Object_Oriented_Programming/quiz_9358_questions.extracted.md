## Q1
Given the following class definition, which of the following correctly sets the name of the person to "John"?

- a) Person p = new Person(); p.setName("John");
- b) Person p = new Person(); p.name = "John";
- c) Person p = new Person(); p.name("John");
- d) Person p = new Person(); p.getName("John");
- Correct answer: Person p = new Person(); p.setName("John");

## Q2
What is a constructor in Java?

- a) A method that is called automatically when an object is destroyed
- b) A method that must be explicitly called to initialize objects
- c) A method that performs arithmetic operations
- d) A special method used to initialize objects
- Correct answer: A special method used to initialize objects

## Q3
What is the purpose of a class in Java?

- a) To perform arithmetic operations
- b) To define a blueprint for objects
- c) To define methods only
- d) To serve as a placeholder for variables
- Correct answer: To define a blueprint for objects

## Q4
Which access modifier allows a class attribute or method to be accessible from any other class?

- a) public
- b) private
- c) default
- d) protected
- Correct answer: public

Which of the following statements correctly creates an instance of a class named Student?
