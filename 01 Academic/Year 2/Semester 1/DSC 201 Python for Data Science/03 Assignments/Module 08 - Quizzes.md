### Question 1
**How can you calculate the correlation matrix for numerical columns in a DataFrame?**

Answer:
- c. `df.corr()`

### Question 2
**To identify the unique values in a column of a Pandas DataFrame, you can use:**

Answer:
- d. `df['column'].unique()`

### Question 3
**What does the `df.describe()` method in Pandas provide?**

Answer:
- c. Statistical summary of numerical columns.

### Question 4
**Which method can be used to identify duplicate rows in a DataFrame?**

Answer:
- c. `df.duplicated()`

### Question 5
**Which method can be used to identify duplicate rows in a DataFrame?**

Answer:
- b. `df.duplicated()`
