# Foundation - Data Types, Variables, and Operators Quiz

**Source Reference**: [[Source - Foundation - Data Types Variables and Operators Quiz.txt]]

## Question 1
**What will be the output for this code? `#print("The following is")`**

**Answer: b. Nothing**

The line starts with `#`, so it is a comment. Python ignores it during execution.

## Question 2
**What will be the output of the following code?**

```python
string1 = "Malaba"
string2 = " "
string3 = "Kenya"
print(string1 + string2 + string3)
```

**Answer: a. Malaba Kenya**

The `+` operator concatenates the three strings, including the space stored in `string2`.

## Question 3
**What data type is the result of the expression `3 + 4.0`?**

**Answer: d. float**

Adding an integer and a float gives a float result.

## Question 4
**What does the following code output: `print(type([1, 2, 3]))`?**

**Answer: b. `<class 'list'>`**

Square brackets create a list in Python.

## Question 5
**What is the correct way to assign the integer value `10` to a variable named `x` in Python?**

**Answer: d. `x = 10`**

Python assignment places the variable name on the left and the value on the right.

## Question 6
**What is the output of `type("Hello")`?**

**Answer: a. `<class 'str'>`**

Text in quotes is a string.

## Question 7
**What is the result of `type(None)`?**

**Answer: d. `<class 'NoneType'>`**

`None` is Python's special value for no value.

## Question 8
**What will be the result of the expression `type(True)`?**

**Answer: b. `<class 'bool'>`**

`True` and `False` are Boolean values.

## Question 9
**Which of the following is a valid variable name in Python?**

**Answer: d. `variable_name`**

Underscores are allowed in Python variable names. Spaces, hyphens, and names starting with digits are not valid.

## Question 10
**Which of the following is not a built-in data type in Python?**

**Answer: a. `map`**

`list`, `tuple`, and `set` are built-in collection data types. `map` is a built-in function, not a data type.

## Question 11
**Which of the following statements will create a string variable?**

**Answer: a. `name = 'Alice'`**

The value is enclosed in quotes, so Python stores it as a string.
