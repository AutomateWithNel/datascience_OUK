# Foundation - Introduction to Python

## Source
- [[../../01 Source Materials/Foundation - Introduction to Python/DSC 201 - Foundation - Introduction to Python.pdf]]

## Key Ideas
- Python is a free high-level programming language with readable syntax.
- A high-level language is closer to human language and can run across platforms such as Windows, Linux, and macOS.
- Python has many libraries, which makes it useful for academic work, scientific computing, machine learning, statistics, and data analysis.
- A program is a set of instructions that tells a computer what to do.
- Programming, also called coding, is the act of writing instructions for a computer.
- Programming languages are used by humans to write instructions, but computers ultimately understand machine language.
- Machine language is a low-level language made of instructions the computer can directly understand.

## Development Environment
- `Compiler`: translates a whole high-level program into machine language.
- `Interpreter`: translates and runs code line by line.
- `Editor`: software used to write and edit code.
- `IDE`: an integrated development environment that combines tools such as an editor and a translator for writing and running programs.

## Components of a Python Program
- `Statement`: an instruction that the Python interpreter can execute.
- `Comment`: an explanation written by the programmer to make code easier to read. Python ignores comments when running the program.
- Single-line comments begin with `#`.
- Multi-line comments can be written using triple quotes.
- `Indentation`: spacing at the beginning of lines. Python uses indentation to organize blocks of code.

## What To Remember For Exam Use
- Python is called high-level because it is closer to human language than machine language.
- The interpreter runs Python code line by line.
- Comments improve readability but are ignored by the Python interpreter.
- Indentation is part of Python syntax and becomes especially important in loops, conditions, and branching.
