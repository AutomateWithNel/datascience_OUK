Module 5: Data wrangling
This module delves into the essential concepts and techniques of data wrangling using Pandas. You will learn how to perform various data wrangling tasks, such as joining, combining, reshaping, filtering, and reducing data. At the end of the module, you are expected to have acquired the skills necessary to clean, reshape, and prepare datasets for analysis.
Learning outcomes
Indetify the different data wrangling operations available in Pandas
Explain the purpose and functionality of data joins, reshaping techniques, and filtering methods in Pandas.
Apply data wrangling techniques to real-world datasets using Pandas
Evaluate the effectiveness of different data wrangling techniques in achieving specific data manipulation objectives
Learning content
Data wrangling: Join; combine; reshape; filter; data reduction
Learning activities
WHY DATA WRANGLING
In real life applications, data comes from different sources in different formats. There is always a need to rearrange, combine, reshape, filter the data before carrying out any analysis on it. In this section we are going to be looking at the techniques for carrying out this particular tasks.
What is data wrangling
The process of changing, or transforming data from one particular shape to another particular shape is usually reffered to us data wrangling. Some people refer to this process as data mugging. Data wrangling involves restructuring, cleaning and converting it into a format that makes it easy to analayse the data.
Combining dataframes
: Pyhton offers several methods for combining dataframes. You can combine dataframes using the merge() function, the join function and the concatenate function.
Join
Sometimes you might have data in different files. You can load the data into two different dataframes and use the join method to combine the two dataframes. The join function is used when you have a common column or a common index. It is more easy to use the join function using an index rather than a column.
The syntax for the join method is as follows;
DataFrame.join(other, on=None, how='left', lsuffix='', rsuffix='', sort=False)
Parameters
other
: This is the name of the the other dataframe to be combined
how
: This indicates the type of the join to be performed. There are different types of joins that will result in different dataframes.
left
: Use the keys found in the first dataframe. Get all the things from the left dataframe and only the matching values from the right(other) dataframe
right
: Use the keys found in the other dataframe. Gets all the values from the right dataframes and only those that match in the right daaframes.
outer
: Uses the keys from both dataframes, returns both the elements in both dataframes
inner
: Uses the intersection of keys from both dataframes, it returns only the data elements that are have the same keys in both dataframes
sort
: Indicates if the resultant dataframes will be sorted. A value of "false" will indicate that the dataframe will not be sorted and a value of "True" will sort the resultant dataframe.
rsuffix:
This is the suffix to be used for overlapping columns

We can use the join function to combine the two dataframes to form one dataframe with the the list of all the products purchased by the different customers.

Pivoting
Pivoting is a data transformation process commonly used in data analysis to reshape data. In pandas, pivoting involves converting a DataFrame from a long or tall format (where rows represent individual observations) to a wide format (where each unique value in a column becomes a new column). This is useful for creating summary tables and for visualizing data more effectively.
Pivot:
This creates a new DataFrame by rearranging the data such that unique values from one column become the columns of the new DataFrame, and values from another column become the rows.
Pivot Table:
This is Similar to a pivot, but it allows for the aggregation of data. For example, it can sum, average, or count values within each group of the new table.

Stacking
Stacking is another data transformation process in pandas, used to convert a DataFrame from a wide format to a long format. It's the opposite of unstacking or pivoting, where you might go from a long format to a wide format. Stacking is particularly useful when you need to perform operations on data that is easier to handle in a long format or when preparing data for certain types of analyses or visualizations.
There are two main concepts in stacking: Stacking and Unstacking
Stacking
: Stacking reshapes the dataframe by converting the data into a stacked form. This involves creating a pivot of the innermost column index into the innermost row index.

We can select a portion of the columns or a portion of the dataset based on a condition applied to a column.

Key things to note
The query() method only works if the DataFrame’s column names are valid Python identifiers (no spaces or special characters). If not, use backticks to enclose column names.
When using variables inside the query string, prefix them with @.
String operations like .str.contains() can be used within query() by referring to the column as a string attribute.
