Aggregation and group operations

Aggregation and grouping are important techniques that can be used for summarizing, transforming and filtering datasets. Aggregation and grouping make it possible to have efficient analysis for large datasets by grouping data based on a certain creteria. This operations usually helpful for reporting and data visualization.

Pandas provides various ways that can enable you to create slices, dices summaries of a dataset in a very natural way.

Groupby mechanics 

Group operations follow the split-apply-combine process as coined by Hadly wickman. In the first process the data cointained in pandas object either dataframe or series is split into different groups based on the different keys that you will provide.Then a function is applied to the groups after which the results are combined to the original datarame.

Aggregation : Aggregation involves the process of computing/ calculating summary statistics for a given set of values

Without the aggeregate functions, the groupby() function does not provide any output. The groupby() function works with the aggregation functions. The aggregation functions do culculations on the grouped dataset providing more insights into the data and helps in understading the data better.

The following are some of the predifined aggregate functions that you can use 

count(): Counts the number of non-NA values in the group
sum(): Sum of non-NA values
mean: Calculates the Mean of non-NA values
median: Gets the arithmetic median of non-NA values
std: Calculates the var Unbiased (n – 1 denominator) standard deviation and variance
min, max: Gets the Minimum and maximum of non-NA values
prod: Calculates the product of non-NA values
first, last: Gets the First and last non-NA values

To aggregate the data using the descrptive statistics we can use the aggregate fuction agg() with the desired function such as the "mean" and "std". We can use the grouped object we created in the previous section called data_two to demonstrate this as follows

Removing the indexing 

In pandas dataframe, each row is usually assigned an index for indefification purposes. Sometimes, you might want to remove this indexes or change the indexing especially if you are doing data cleaning and preprocessing. 

In pandas an index is a lebel that indetifies each row in a dataframe o a series. When loading your data, if you do not specify the index range pandas assigns a range of index starting from 0 to the total count of the rows in the dataframe -1. You can create you own custom index such as the date, names and so on.

Resetting an index in pandas 

In some instances when you have done some filtering or modification on your dataframe, there might be some issues such as gaps or jumbling up of the indexes in such cases, you will need to reset the index. Pandas provides a function called reset_index() that will enable you to reset the index of the dataframe.

The syntax for reseting the dataframe is as follows

DataFrame.reset_index(drop=False, inplace=False)
