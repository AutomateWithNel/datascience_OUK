Bar graph: To create a bar graph, we will use the bar() method

import matplotlib.pyplot as plt

# Example dataset
months = ['January', 'February', 'March', 'April', 'May', 'June', 
          'July', 'August', 'September', 'October', 'November', 'December']
temperatures = [0, 2, 5, 9, 14, 18, 22, 21, 17, 11, 5, 1]

# Create a figure and axis
fig, ax = plt.subplots(figsize=(10, 6), dpi=100)

# Create a bar graph
ax.bar(months, temperatures, color='skyblue')

# Add titles and labels
ax.set_title('Average Monthly Temperature')
ax.set_xlabel('Month')
ax.set_ylabel('Temperature (°C)')

# Rotate month labels on x-axis to 90 degrees for better readability
plt.xticks(rotation=90)

# Display the bar graph
plt.show()

Boxplot

A box plot (or box-and-whisker plot) is a way of displaying the distribution of data based on a descriptive statistics such as the minimum, first quartile (Q1), median, third quartile (Q3), and maximum. Box plots are usually used to identify outliers and show overall distribution and spread of the data.

The syntax for the boxplot is 

    matplotlib.pyplot.boxplot(x, notch=None, sym=None, vert=None, whis=None, positions=None, widths=None, patch_artist=None, bootstrap=None, usermedians=None, conf_intervals=None, meanline=None, showmeans=None, showcaps=None, showbox=None, showfliers=None, boxprops=None, tick_labels=None, flierprops=None, medianprops=None, meanprops=None, capprops=None, whiskerprops=None, manage_ticks=True, autorange=False, zorder=None, capwidths=None, label=None, *, data=None)[source]

Violine plot: Violine plot indicate the data distibution for more than one group by making use of density plots. The syntax for creating a violine plot in matplotlib is as follow

    matplotlib.pyplot.violinplot(dataset, positions=None, vert=True, widths=0.5, showmeans=False, showextrema=True, showmedians=False, quantiles=None, points=100, bw_method=None, side='both', *, data=None)

Violine plot is used for

  1. showing the shape of the data distribution, including the probability density, which helps in understanding the spread and concentration of the data.
  2. Symmetry and Skewness: They make it easy to see if the data is symmetrical or skewed, and if there are any multimodal distributions (i.e., distributions with multiple peaks).
  3. Displaying ranges and outliers in the data

Seaborn is a statistical library that is based on matplotlib and it integreates very easily with the pandas library. Seaborn provides the functionality of creating the various plots. It provides a high-level interface for drawing attractive and informative statistical graphics. It is well-suited for visualizing complex datasets and statistical relationships.

Seaborn features

  1. It provides a simpler and more intuitive interface for creating plots.

  2. It provide easy handling of data structures like DataFrames.

  3. It provides default themes and color palettes that are aesthetically pleasing and informative.

  4. It enables the creation of complex visualizations with minimal code.

We can now use plotily to draw a scatter plot. A scatter plot shows relationship between variables

fig= px.scatter(data, x="sepal_width",y="sepal_length", color="species")

We can also do a box plot to see if we have outliers in our sepal_width. An outlier is a datapoint that does not fit within the normal range of values, it deviates from the other data points in the dataset.

We can also create a line graph to check if there is a relationship between the petel width and the legth
