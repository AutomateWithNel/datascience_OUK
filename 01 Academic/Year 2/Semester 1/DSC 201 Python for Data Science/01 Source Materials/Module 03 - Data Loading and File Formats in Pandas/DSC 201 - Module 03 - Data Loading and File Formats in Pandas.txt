Reading the excel file in pandas To read excel file in pandas you will use the pd.read_excel(). The read_excel() function expects the following parameters. Excel file path : Provides the path to the excel file. Sheet_name : This specefies the sheet name that your data is in this is applicable in cases where the excel file cointains more than one sheet. Header : The header indicates the row to be used for the names of the column. The default one is the row at index 0. usecols : Indicates the columns that you will need to use in your dataframe. The default is none which will load all the columns in the dataframe. If you need only specific columns from your excel sheet, then you need to specify them as the list. The list above is not exhostive of all the parameters that you can set when reading an excel file. It is widely used in scientific computing and data-intensive applications because it supports the creation, access, and sharing of complex data. Key features of HDF5 include: Hierarchical Structure: Data is organized in a file in a hierarchical manner, similar to a filesystem. This allows the grouping of related datasets and metadata in a structured way. Scalability: HDF5 can efficiently handle very large datasets, both in terms of size and complexity. It is optimized for high performance in both reading and writing operations. Portability: HDF5 files are designed to be platform-independent, meaning they can be created and read on different types of systems without compatibility issues. Flexibility: It supports various data types and complex data relationships, including multidimensional arrays, tables, and complex objects. This makes it suitable for a wide range of applications. Metadata Storage: HDF5 allows the storage of rich metadata along with the data, which is crucial for documenting and understanding the data. Compression: The format supports compression and other storage optimizations to save space and improve performance. Parallel I/O: HDF5 provides support for parallel I/O, allowing multiple processes to read/write data simultaneously, which is important for high-performance computing environments. HDF5 is commonly used in fields such as astronomy, physics, engineering, bioinformatics, and many other scientific domains where managing large and complex datasets is essential. It is supported by various programming languages, including Python (with the h5py library), C, C++, Fortran, and others. Pandas provides the read_hdf() for reading HDF5 files. Interacting with web APIs In some cases, organizations provides access to data using an API. An API in full is Application Programming Interfaces. It provides for an interface for applications to be able to communicate with one another. There are different types of APIs for different purposes Web APIs : Web APIs are APIs that provide transfer of machine readable data between web based applications. There are different formats used in web APIs. This section reviews a few of them REST (Representational State Transfer): Uses HTTP requests to GET, POST, PUT, DELETE data. It's stateless and often returns data in JSON or XML format. SOAP (Simple Object Access Protocol): A protocol for exchanging structured information in the implementation of web services. It uses XML and can operate over various protocols such as HTTP, SMTP. GraphQL: Developed by Facebook, it allows clients to request exactly the data they need. It provides a more flexible and efficient alternative to REST. Example In this example we are accessing the weather data provided by the open weather map using a REST API.
