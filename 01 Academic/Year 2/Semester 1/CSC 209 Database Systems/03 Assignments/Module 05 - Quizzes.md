### Question 2

Which of the following does LEAST describes SQL

Options:
a. It is only used in relational databases
b. It is divided into three types namely DCL, DDL and DML.
c. It is a query language, meaning it is for querying (asking) data

### Question 3

Which of the following is NOT a valid join SQL statement

Options:
a. Select item_name, price FROM items inner join suppliers ON suppliers.item_name= items.item_name
b. Select items .item_name, items .price FROM items inner join suppliers ON suppliers.item_name= items.item_name
c. All of the above
d. Select item_name, price FROM items inner join suppliers ON suppliers.item_name= items.item_name WHERE item_name=’fff’
