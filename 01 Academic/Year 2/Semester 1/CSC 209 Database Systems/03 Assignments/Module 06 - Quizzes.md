## Module 6 Quiz

### Question 1

SQL can execute it in two different way, which are these two ways

Options:
a. Full table scan and querying
b. Partial Indexing and partial scanning
c. Indexing and partial scanning
d. Full table scan and indexing

Answer: Full table scan and indexing

### Question 2

What do you understand by query planning?

Options:
a. How and what the query does
b. What the query does
c. How the query execution occurs
d. What, why and how the query executes

Answer: How the query execution occurs

### Question 3

Which of the following is not an operation of relational algebra?

Options:
a. Set Difference
b. Union
c. Query join
d. Rename

Answer: Query join

### Question 4

Which of the following is true about query planning

Options:
a. Query planning can consume computing resources
b. Query planning affects query performance
c. Query planning dictates the syntax of a query
d. None of the above

Answer: Query planning affects query performance
