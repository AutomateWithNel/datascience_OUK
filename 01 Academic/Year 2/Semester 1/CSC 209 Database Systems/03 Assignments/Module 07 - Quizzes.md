## Module 7 Quiz

### Question 1

SQL is a database programming language.

Options:
a. True
b. False

Answer: True

### Question 2

What is the difference between PHP and SQL

Options:
a. SQL is structured query language while PHP is client side programming language
b. All of the above
c. PHP is client side scripting language while SQL is server side
d. PHP is a server side language while SQL is a database embedded language

Answer: PHP is a server side language while SQL is a database embedded language

### Question 3

What is the following least describes the purpose of Database programming languages?

Options:
a. To display data extracted from the database
b. To replace database query language
c. To transmit input from users to the database
d. To extract data and information efficiently from the database

Answer: To replace database query language
