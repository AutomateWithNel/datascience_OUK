## QUIZ 2

### Question 1

For a relation to be in second normal form, it must be

Options:
a. In 2NF and have a single Column Primary Key
b. In 1NF and have a single Column Primary Key
c. In 1NF and have a Column Primary Key
d. In 2NF and have a composite Column Primary Key

Answer: In 1NF and have a single Column Primary Key

### Question 2

The process of database normalization is a set of rules called

Options:
a. Normal forms
b. Indexes
c. Relations
d. Tables

Answer: Normal forms

### Question 3

Using entity relationship diagram, which one of the following is NOT true about a possible relationship between an item/product entity and a supplier entity

Options:
a. A supplier may not supply any item
b. A supplier supplies items
c. An item can be supplied by several suppliers
d. One supplier can supply only one item

Answer: A supplier may not supply any item

### Question 4

When representing an entity in an entity relationship diagram, we use

Options:
a. Square
b. Rectangle
c. Eclipse
d. Oval

Answer: Rectangle

### Question 5

Which of the following statements is correct

Options:
a. An ER diagram is made up of relations only
b. An entity is made up of attributes and relations
c. The attributes of an entity define the characteristics of that entity
d. Entity relations can happen even without an entity

Answer: The attributes of an entity define the characteristics of that entity
