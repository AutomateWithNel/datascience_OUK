## Knowledge Check

### Question 1

Denormalization of entities can be done during physical database design, true or false?

Options:
a. True
b. False

Answer: False

### Question 2

Logical and physical database design can be the same when done by an expert.

Options:
a. False
b. True

Answer: False

### Question 3

Which of the following statements is not true about physical database design?

Options:
a. Entities get transformed into tables
b. Physical design happens before physical design {answer}
c. Attributes get converted to table columns
d. Primary keys are identified from attributes

Answer: Physical design happens before physical design {answer}

### Question 4

Which of the following steps are correct for a database life cycle?

Options:
a. Physical design, Requirements analysis and logical design
b. Requirements analysis, logical design and physical design
c. All of the above.
d. Logical design, Requirements analysis, and physical design

Answer: Requirements analysis, logical design and physical design
