## General Quiz

**What does the term "encapsulation" refer to in OOP?**

**What is Object-Oriented Programming (OOP)?**
*   **a. A programming paradigm based on objects and classes.**

**Which of the following is a benefit of using OOP?**
*   **a. Code reuse through inheritance**

**Which of the following is NOT a principle of OOP?**
*   **d. Compilation**

**Which statement best describes the main difference between procedural programming and object-oriented programming?**
*   **a. OOP organizes code around objects and classes, while procedural programming organizes code around functions.**
